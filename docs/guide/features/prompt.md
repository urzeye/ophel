# ⌨️ 提示词助手

提示词助手是 Ophel 的核心功能之一，帮助你高效管理和使用常用提示词，让优质 Prompt 触手可及。

## 功能概览

<div class="tip custom-block">

**适用场景**

- 📝 保存和复用常用提示词
- 🔄 使用变量模板创建动态提示词
- 📂 分类整理不同场景的提示词
- ⚡ 快速访问最近使用的提示词

</div>

## 高级特性

### 变量支持

使用 `{{变量名}}` 语法定义动态变量，使用提示词时会自动弹窗填写：

**示例：翻译助手**

```markdown
你是一位专业的翻译专家。请将以下内容翻译成{{目标语言}}：

{{待翻译内容}}

要求：
- 保持原文风格
- 译文流畅自然
- 专业术语准确
```

**使用时：**

1. 点击使用该提示词
2. 弹窗提示填写变量：
   - 目标语言：`英语`
   - 待翻译内容：`你好，世界！`
3. 自动生成最终提示词并插入输入框

**变量高级用法：**

| 语法 | 说明 | 示例 |
|------|------|------|
| `{{变量}}` | 基础变量 | `{{主题}}` |
| `{{变量:默认值}}` | 带默认值 | `{{语言:中文}}` |
| `{{变量:选项1或选项2}}` | 下拉选择 | `{{风格:正式或轻松或幽默}}` |

### Markdown 预览

编辑提示词时，右侧实时预览渲染效果：

- 📝 代码块语法高亮
- 📋 列表正确缩进
- 🔗 链接可点击
- 💡 变量高亮显示

### 分类管理

为提示词创建分类，系统自动分配色彩：

```
📁 工作效率
├── ✍️ 文案润色
├── 📊 数据分析
└── 📧 邮件撰写

📁 编程开发
├── 🐛 Debug 助手
├── 📖 代码解释
└── ⚡ 代码优化

📁 学习助手
├── 📚 概念解释
├── 🎯 习题解答
└── 📝 笔记整理
```

**分类功能：**

- ➕ 创建自定义分类
- 🎨 自动分配分类颜色
- 📂 拖拽移动提示词到分类
- 🔍 按分类筛选

## 数据管理

### 快捷访问

#### 置顶常用

将最常用的提示词置顶，一眼就能找到：

- ⭐ 点击星标图标置顶/取消
- 📌 置顶的提示词显示在列表最前
- 🔢 支持置顶多个提示词

#### 最近使用

自动记录使用历史，快速访问最近用过的提示词：

- 🕐 按使用时间倒序排列
- 📊 显示使用次数统计
- 🔄 一键清空历史记录

### 导入/导出

提示词数据独立管理，支持：

#### 导出

```json
{
  "version": "1.0",
  "exportTime": "2024-01-15T10:00:00Z",
  "prompts": [
    {
      "id": "xxx",
      "title": "翻译助手",
      "content": "你是一位专业的翻译专家...",
      "category": "工作效率",
      "pinned": true,
      "useCount": 42
    }
  ],
  "categories": ["工作效率", "编程开发", "学习助手"]
}
```

#### 导入

- 📥 支持导入 Ophel 格式的 JSON 文件
- 🔀 智能合并：相同标题提示词可选择覆盖或跳过
- 📂 自动创建不存在的分类

## 使用流程

### 创建提示词

1. 打开提示词面板
2. 点击「新建提示词」
3. 填写标题和内容
4. 选择或创建分类
5. 点击保存

### 使用提示词

**方式一：面板选择**

1. 打开提示词面板
2. 点击目标提示词
3. 填写变量（如有）
4. 自动插入输入框

**方式二：快捷键**

1. 在输入框中按 `Alt + P`
2. 搜索或选择提示词
3. 确认使用

**方式三：斜杠命令**

1. 在输入框中输入 `/`
2. 出现提示词搜索框
3. 输入关键词筛选
4. 选择并使用

## 内置提示词

Ophel 预置了几个实用的提示词模板：

| 名称 | 用途 |
|------|------|
| 🌐 翻译助手 | 多语言翻译，保持原文风格 |
| ⚡ 代码优化 | 优化代码质量和性能 |
| 📝 文案润色 | 优化文字表达 |
| 🤔 深度分析 | 全面分析问题 |

::: tip 提示
你可以根据自己的需求修改内置提示词，或以它们为模板创建新的提示词。
:::

## 设置选项

| 选项 | 描述 | 默认值 |
|------|------|:------:|
| 显示使用次数 | 是否在列表显示使用统计 | 开启 |
| 斜杠命令 | 是否启用 `/` 快捷触发 | 开启 |
| 自动关闭面板 | 使用后是否自动关闭面板 | 开启 |

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt + P` | 打开提示词面板 |
| `/` | 在输入框中触发提示词搜索 |
| `Esc` | 关闭提示词面板 |
